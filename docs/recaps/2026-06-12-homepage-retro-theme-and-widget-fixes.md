## Lessons
- **Widgets are honest narrators** вЂ” three "errors" were a dead backend (Immich), an expired key
  (Tailscale), and a per-instance-credential gap (Pi-hole). Diagnose the server path; don't
  blame the dashboard.
- **"Too bright" and "washed out" were the same bug** (low contrast) pulling the font lever
  opposite ways вЂ” the real fix was the background/surface.
- **`op` from automation has a short biometric window** вЂ” re-`op signin` then run the op step
  immediately, batched into one call.
